## Anki Web Cards

Aplicativo web de flash cards para aprendizado. Exibe, adiciona e remove cartões de memória com perguntas e respostas.

## Projeto

- Cria cartões viráveis usando CSS
- Cria sobreposição 'adicionar novo cartão' com formulário
- Exibe cartões de pergunta e vira para resposta
- Visualiza cartões anterior e próximo
- Adiciona novos cartões em local storage (armazenamento local)
- Remove todos os cartões do local storage
