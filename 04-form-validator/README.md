## Validador de Formulário 

Validação simples de formulário no lado do cliente. Verifica campos obrigatórios, comprimento, e-mail e correspondência de senha.

## Projeto

- Criar interface do formulário
- Mostrar mensagens de erro abaixo de entradas específicas
- checkRequired - para verificação de campos obrigatórios
- checkLength - para verificar comprimento mínimo e máximo
- checkEmail - para validar e-mail com regex
- checkPasswordsMatch - para verificação de senha de confirmação
