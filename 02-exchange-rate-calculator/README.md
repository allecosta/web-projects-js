## Exchange Rate - BETCOIN

Selecione países para obter a taxa de câmbio para um valor específico.

## Projeto

- Exibir interface com 2 listas de seleção para países e 2 campos de entrada para valores
- Obter taxas de câmbio da API https://api.exchangerate-api.com
- Exibir os valores para ambos os países
- Atualizar valores quando o valor for alterado
- Trocar taxas dos países
