## Custom Vídeo Player - We Are The Wave

Player de vídeo personalizado usando o elemento de vídeo HTML5 e API JavaScript.

## Projeto

- Exibir player de vídeo estilizado com CSS
- Reproduzir/pausar/Parar
- Barra de progresso do vídeo
- Definir tempo da barra de progresso
- Exibir tempo em minutos e segundos
