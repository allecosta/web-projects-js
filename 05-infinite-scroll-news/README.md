## Infinite Scrolling & Filter - The Times News

Exibe posts via API [jsonplaceholder](https://jsonplaceholder.typicode.com), com rolagem infinita, buscando posts e também caixa de filtro.

## Projeto

- Interface e animação personalizada de carregamento em CSS
- Busca posts iniciais da API e exibe
- Ao rolar para baixo, mostra carregador e busca o próximo conjunto de posts
- Filtro para os posts buscados
